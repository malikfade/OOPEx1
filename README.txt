

We have done a project that allows the user to create functions.. implement arthimetic operations on them and use them however he wants and ofcourse draw them. 
We have Four different sections :  MONOM, POLYNOM , FUNCTION_GUI and COMPLEX_FUNCTION .


Polynom - Polynom is a series of monoms applied together.
Monom - A function that looks like ax^b.

ComplexFunction Its an Operation of a function , a function can be a monom..polynom.. or ofcourse the complex function.

Function_GUI - its the class that can represent the functions .. read and write and ofcourse draw them on gui.
